Visual identity chosen: "Modern Polish" direction.

- **Primary:** `#1E3A5F` (deep navy) — trust, professionalism
- **Accent:** `#E8683A` (warm coral) — energy, CTAs, highlights
- **Background:** `#FAFAFA` / **Foreground:** `#111827`
- **Muted:** `#F4F4F5` / **Muted fg:** `#6B7280` / **Border:** `#E5E7EB`
- **Success:** `#059669` / **Destructive:** `#DC2626`
- **Typography:** IBM Plex Sans (kept), IBM Plex Mono for code
- **Mode:** Light-only, no dark mode

**Why:** Navy communicates trust (handles money), coral is distinctive and energetic. Works for retreats, workshops, and any group event — no brand ceiling. Neutral host brand lets organizer brandColors layer on top well.

**How to apply:** All UI work should use these colors via CSS variables. Accent color for CTAs and highlights. Primary for navigation and key UI elements.
